# CAPEC™ 702: Exploiting Incorrect Chaining or Granularity of Hardware Debug Components

## Description

{'p': {'__prefix': 'xhtml', '__text': 'An adversary exploits incorrect chaining or granularity of hardware debug components in order to gain unauthorized access to debug functionality on a chip. This happens when authorization is not checked on a per function basis and is assumed for a chain or group of debug functionality.'}}

Source: [CAPEC™ 702](https://capec.mitre.org/data/definitions/702.html)
