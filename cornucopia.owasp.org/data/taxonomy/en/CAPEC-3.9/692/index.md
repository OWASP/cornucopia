# CAPEC™ 692: Spoof Version Control System Commit Metadata

## Description

{'p': {'__prefix': 'xhtml', '__text': "An adversary spoofs metadata pertaining to a Version Control System (VCS) (e.g., Git) repository's commits to deceive users into believing that the maliciously provided software is frequently maintained and originates from a trusted source."}}
